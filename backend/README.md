# RentFlow Backend ⚙️

This is the backend service for the RentFlow Boarding House Management System, built with **Node.js** and **Express**.

## 🚀 API Responsibilities
- **Authentication**: Managing user logins and session states.
- **Tenant Management**: Handling resident profiles and room allocations.
- **Billing Engine**: Calculating dues and processing payment history.
- **Maintenance Tracking**: Managing repair requests and status updates.

## 🛠️ Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   node server.js
   ```

For more information about the full system, please see the [Root README](../README.md).
